> 文章主要记录如何利用Echarts实现需求

- 多Y轴

```
// series 通过 yAxisIndex 字段来确定是属于哪一个y轴

grid: {
    show: true,
    left: '3%',
    right: '10%',
    bottom: '5%',
    containLabel: true,
},

yAxis: [
    {
        type: 'value',
        name: '油压(MPa)',
        boundaryGap: true,
    },
    {
        type: 'value',
        name: '伸长量(mm)',
        boundaryGap: true,
    },
    {
        type: 'value',
        name: '张拉力(KN)',
        boundaryGap: true,
        offset: 120, 
        // 通过offset偏移是得多根y轴不重叠， 可以通过grid字段来设置图的大小，让y轴有地方显示
    }
],
```

- 给指定区域上色

上色可以通过`markArea`来实现

```
series: [
    {
        name: '张拉力1',
        type: 'line',
        stack: 'stretch1',
        yAxisIndex: 2,
        data: mockData.map(item => item.stretch1),
        markArea: {
            data: [
                [
                    {
                        name: '第一级张拉',
                        xAxis: 0,
                        itemStyle: {
                            color: '#D4E6EF'
                        }
                    },
                    {
                        xAxis: (mockData1.length / mockData.length) * 100
                    },
                ],
                [
                    {
                        name: '第二级张拉',
                        xAxis: (mockData1.length / mockData.length) * 100,
                        itemStyle: {
                        color: '#DDDDEE'
                        }
                    },
                    {
                        xAxis: 200 // 100并不能完全覆盖整个图表，所以用了200

            ]

    },
```
